# .about
- `evegr2tojson.exe` - A command line tool for converting eve online gr2 files to json.
- `blender_gr2_loader.py` - A blender 5.0 plugin for loading `.gr2` and `.gr2_json` file.

# .usage (windows)
`evegr2tojson.exe <src> <dest>`
